How to Play

It is not as intuitive as it should.

Main Menu

The main menu of ARS is brought up at any moment using the Sprint + Interact keys, Shift+E on keyboard.

This menu allows you to select a track, the lap number, the opponent vehicles, and how many opponents will participate in the race.

  • The first item is the Track Selection. It will let you input a few keywords, separated by spaces. These will be used to search for tracks that fit the keyword criteria. On the same item you can select the current filtered track with the arrow keys.

  • Disciplines lets you input some keywords to filter what cars will be summoned. muscle street will, for example, select any car which is considered a Muscle car, or has a street tag on it.
    The input has a few modifiers to better choose what kind of vehicles you want summoned.
    • +tag will mean that tag is required. All cars summoned need to have this tag.
    • -tag will mean that tag is prohibited. All cars summoned cannot have this tag.
    • tag will mean that tag is optional. If a car contains this tag, it will be included.
    You can freely include as many tags as you want, with one modifier on each, or none.

  • Grid Size allows you to decide how many AI racers will participate in the race. You can enter a number here, or use the arrow keys to modify it.

  • Laps does the same, but regarding how many laps the race will consist of. You can enter a number here, or use the arrow keys to modify it too.

  • Quick Start loads the race itself. It will first load the track, then the vehicles, and place all of you in the grid.

    • You can participate in the races by simply being in a vehicle when you hit Start. If you're on foot, you'll not participate.
